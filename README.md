<h1>CounterForTop100</h1>

This app counts the number of participants on the site and sorts of votes. It is needed in order to reveal the bests students

The following libraries were used in the project:
<br>
<p>-request</p>
<p>-beautyfullsoup</p>
<p>-tkinter</p>
